Shelve-It


Shelve-It is a group project done at Southern Methodist University bootcamp. The team members are Justin Cheng, Emir Velasqez, Jhalya Young, and Hibah Shafi. This is our first full stack project and the languages and technologies used are the following: HTML, CSS, Bootstrap, JavaScript, jQuery, Node Express, MySQL, SQL, ORM, CRUD, MVC, quill editor, After Effects, Firebase DB, etc. Shelve-It mission is to create a new internet community to support low-ley authors.

Shelve-It is a storytelling platform for amateur authors and eager readers. For readers, it consists of a digital library and the ability to search a book using Google API. For authors, it is a free online writing site that provides the tools for writers to publish their work.
